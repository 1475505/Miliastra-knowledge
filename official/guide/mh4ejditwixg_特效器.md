# 一、什么是特效器

_特效器_是一组以时间轴方式编辑的特效/音效组合体，奇匠可以采用与普通特效相同的方式进行播放或挂载

# 二、特效器编辑入口

如下图所示，进入战斗预设页签，选择特效器，点击【新建特效器】

# 三、特效器的基础概念

## 1.特效器的分类

与普通特效相似，特效器也分为限时和持续两种。

_限时特效器_为有限时长的特效器，在播放到达_持续时长_时，会自行销毁。限时特效器仅能使用【播放限时特效】节点播放

_持续特效器_为无限时长的特效器，不会自行销毁。但与持续特效相似，可以在节点图中用特效资产配置ID或特效实例ID进行销毁。持续特效器仅能使用【挂载循环特效】节点进行挂载

注意，在特效器创建时就需要确定特效器的类型，之后无法更改

在创建了限时特效器后，可以进入编辑界面调整其持续时长

## 2.特效轨道

特效器可以由多条特效轨道共同组成，点击【新建轨道】可以增加轨道

每条特效轨道上可以任意配置特效或音效。运行时，会沿着特效时间轴，在配置的时间点播放特效或音效

点击对应轨道可以对该轨道的属性进行设置

### **(1)可见性设置**

相关配置均影响此特效轨道内的所有特效对各玩家是否可见，任一条件不满足都会导致该特效轨道内的特效不可见

|     |     |
| --- | --- |
| 参数 | 说明 |
| _可见阵营列表_ | 考虑特效拥有者和当前本地玩家之间的阵营关系：<br>敌对阵营：特效拥有者与本地玩家为敌对时，本地玩家可见到该轨道的特效<br>友善阵营：特效拥有者与本地玩家为友善时，本地玩家可见到该轨道的特效<br>自身阵营：特效拥有者与本地玩家为相同阵营时，本地玩家可见到该轨道的特效<br>默认值为全选，即所有玩家均可看到该轨道上的特效 |
| _本地过滤器_ | 特效轨道会按照一个固定频率在每个玩家本地更新本地过滤器，当返回为是时，该特效可见，返回为否时，该特效不可见 |

### **(2)循环节配置**

|     |     |
| --- | --- |
| 参数 | 说明 |
| _启用循环节_ | 启用时，该轨道变为循环节轨道。也可以在【新建轨道】时直接新建循环节轨道 |
| _循环节起点(s)_ | 表示该循环节在时间轴上的起始位置 |

| _循环次数_ | 该循环节的循环次数，不启用无限循环时该参数可配置 |

## 3.特效(音效)块

在特效轨道上的任意位置，均可配置特效(音效)块。每个特效(音效)块可以引用一个特效资产(音效资产)，并配置其在轨道中的时长(生命周期)

特效资产可以选择限时特效或持续特效

### **(1)特效(音效)块的生命周期配置**

持续播放为是时，该特效会持续播放直到其播放完毕（限时特效），或直到该特效器被销毁。

持续播放为否时，需要配置时长，该特效会在播放该时长后销毁。但如果特效器在其播放时长结束前被销毁，该特效仍然会一起销毁

选择完特效资产并点击【设置为资产时长】后，可以将该特效的时长设置为该特效资产的时长（但由于特效的实际播放时长有一定的随机性，不能完全保证特效表现不会被提前回收）

### **(2)特效的销毁/隐藏时行为**

普通特效和预警特效均可以配置其在被销毁/隐藏时的行为

_销毁时行为_：当特效器被销毁时（例如通过节点图移除、特效器的拥有者被销毁等），该特效资产的行为

_隐藏时行为_：当特效器被隐藏时（例如单位状态-隐藏角色会导致拥有者上挂载的所有特效器被隐藏），该特效资产的行为

该选项的可配置内容取决于选用的特效资产类型：



**限时特效**

是否存在销毁时行为为是时，销毁时行为可选：停止粒子发射/停止并清除特效

_停止粒子发射_：停止当前特效的粒子发射，但已发射的粒子会持续直到生命周期结束，通常用来实现一些特效的自然消散效果。注意，该选项仅对带有粒子的特效资产有意义，如果该特效不包含粒子，则该选项与停止并清除特效效果相同

_停止并清除特效_：停止当前特效的粒子发射，并移除所有特效表现，可以用来将特效表现瞬时清除

是否存在销毁时行为为否时，该限时特效在特效器销毁时不受到影响，会继续按照其生命周期配置完成播放

特殊说明：以上提到的特效触发销毁时，都是尝试销毁。当特效为限时特效且配置是否存在销毁时行为为否时，该限时特效会销毁失败，所以会出现上述在特效器销毁时不受影响继续按照其生命周期配置完成播放的情况

隐藏时行为可选：无/销毁特效

_无_：当特效器被隐藏时，不受到影响

_销毁特效_：当特效器被隐藏时，销毁此特效。当特效器恢复显示时，也不会继续或重播该特效



**循环特效**

必须要配置一种销毁时行为：停止粒子发射/停止并清除特效

_停止粒子发射_：停止当前特效的粒子发射，但已发射的粒子会继续直到生命周期结束，通常用来实现一些特效的自然消散效果。注意，该选项仅对带有粒子的特效资产有意义，如果该特效不包含粒子，则该选项与停止并清除特效效果相同

_停止并清除特效_：停止当前特效的粒子发射，并移除所有特效表现，可以用来将特效表现瞬时清除

隐藏时行为可选：无/同步隐藏特效

_无_：特效器被隐藏时，不受到影响

_同步隐藏特效_：当特效器被隐藏时，此循环特效也被隐藏。当特效器恢复显示时，该特效会恢复显示

### **(3)普通特效块**

在轨道上右键选择【添加事件】，再选择【特效】即可添加普通特效块

选择不同的特效资产，可配置参数会有少许的不同

|     |     |
| --- | --- |
| 参数 | 说明 |
| _特效资产_ | 可以任意选择普通特效资产（限时/循环均可） |
| _播放特效默认音效_ | 为是时，该特效播放时会同步播放该特效自带的默认音效；为否时，则不会播放默认音效 |
| _附加元素属性_ | 仅部分特效有此功能<br>可以将特效表现改为附加元素属性的风格 |
| _特效表现播放速度_ | 仅部分特效有此功能<br>用来调整该特效的播放速度倍率，默认为1（即以原速播放）<br>注意，该播放速度不会影响特效自带的默认音效的播放速度 |
| _偏移_ | 特效通用参数，相对特效器原点的位置偏移 |
| _旋转_ | 特效通用参数，相对特效器原点的旋转偏移 |
| _缩放_ | 特效通用参数，相对特效器的缩放，仅支持等比缩放 |

### **(4)预警特效块**

在轨道上右键选择【添加事件】，再选择【预警特效】即可添加预警特效块

预警特效是一类可以自由配置其静态参数的特效，分为圆形、矩形和扇形三种

预警特效都是限时特效，也可以以限时特效的方式直接调用

# 三、特效器的使用方式

特效器的使用方式与普通特效完全相同，所有可以播放普通特效的地方均可以播放特效器

以节点【播放限时特效】为例：

在特效资产选择框中，可以填入特效器的配置ID

或用搜索框，找到已配置的限时特效器

在特效资产为特效器的情况下，【播放限时特效】节点的功能如下：

|     |     |
| --- | --- |
| **参数** | **说明** |
| _特效资产_ | 选择一个特效器 |
| _目标实体_ | 特效器的拥有者实体 |
| _挂接点名称_ | 特效器的挂接点，即特效器的原点会放置在对应的挂接点位置上 |
| _是否跟随目标运动_ | 决定了特效器内的所有特效是否跟随运动 |
| _是否跟随目标旋转_ | 决定了特效器内的所有特效是否跟随旋转 |
| _位置偏移_ | 特效器的原点相对于挂接点的偏移 |
| _旋转偏移_ | 特效器相对于挂接点的旋转 |
| _缩放倍率_ | 与特效器内的缩放倍率一同作用 |
| _是否播放自带的音效_ | 当此项为是时，特效器内的特效根据自身的【播放特效默认音效】配置决定是否播放自身的默认音效；<br>此项为否时，特效器内的所有特效均不会播放自身的默认音效 |
