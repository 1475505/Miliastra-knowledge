# 一、自定义开关的功能

_自定义开关_可以通过点击或其所映射的自定义变量进行开和关的状态切换，并以此来触发不同的功能逻辑

# **二、自定义开关的编辑**

## **1.添加自定义开关**

在_界面控件组编辑窗口_，添加界面控件模板-自定义开关

### (2)样式

_样式_：支持选择_预设样式、自定义_



**预设样式**

可以选择不同的预设按钮样式，部分预设样式支持自定义图标、颜色、和文本等



**自定义样式**

可以自由配置默认状态、悬停状态、按下状态的素材

|     |     |     |     |
| --- | --- | --- | --- |
| 状态 | 说明 | 样式特征 | 示意图 |
| 开启-默认 | 无交互状态 | / |  |
| 开启-悬停 | 在键鼠操作时，指针悬停在按钮上的状态 | 悬停状态通常用外描边，颜色略微高亮，尺寸略微增大来示意按钮的可交互性 |  |
| 开启-按下 | 在键鼠或触屏操作，按下时的状态 | 按下状态通常用颜色变化，尺寸略微减小来示意按钮处于交互状态，松开后，按钮将返回默认状态 |  |
| 关闭-默认 | 无交互状态 | / |  |
| 关闭-悬停 | 在键鼠操作时，指针悬停在按钮上的状态 | 悬停状态通常用外描边，颜色略微高亮，尺寸略微增大来示意按钮的可交互性 |  |
| 关闭-按下 | 在键鼠或触屏操作，按下时的状态 | 按下状态通常用颜色变化，尺寸略微减小来示意按钮处于交互状态，松开后，按钮将返回默认状态 |  |

### (3)功能



**按键设置**

|     |     |
| --- | --- |
| 配置参数 | 说明 |
| _开关映射自定义变量_ | 支持配置映射玩家的布尔型自定义变量<br>自定义变量变化时，开关会对应变化 |
| _按键映射-键鼠_ | 配置开关对应的PC端快捷使用按键 |
| _按键提示偏移-键鼠_ | 当鼠标悬停开关时，提示信息偏移显示配置 |
| _按键映射-手柄_ | 配置开关对应的手柄快捷使用按键 |
| _按键提示偏移-手柄_ | 当手柄悬停开关时，提示信息偏移显示配置 |
| _打开音效_ | 当开关变成打开状态时，播放该配置音效 |
| _关闭音效_ | 当开关变成关闭状态时，播放该配置音效 |



**响应点击区域**

当且仅当配置样式为自定义时，支持配置点击区域

支持在编辑窗口中直接拖动，调整点击区域

实际触发以此点击区域配置的大小为准

|     |     |
| --- | --- |
| 配置参数 | 说明 |
| _是否可点击_ | 若不开启，则本按钮无点击交互功能 |
| _点击区域预览_ | 开启后，编辑窗口对应控件中浅黄色区域则为点击区域 |
| _区域大小_ | 支持配置跟随按钮尺寸/自定义尺寸 |
| _位置偏移_ | 当 **区域大小** 选择自定义尺寸时，支持配置点击区域的中心位置偏移 |
| _尺寸_ | 当 **区域大小** 选择自定义尺寸时，支持配置点击区域的尺寸 |
